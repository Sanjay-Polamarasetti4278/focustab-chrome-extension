I'm not a software developer. I'm a Data Analyst.

YouTube, Instagram, random browsing. Sound familiar?

So I decided to build something about it. Not just talk about it. Actually build it.

to a fully working Chrome Extension in one day. No coding bootcamp. 
No Stack Overflow rabbit holes. Just clear thinking + the right AI tools.

---

## What FocusTab does

- Full screen wallpaper with a live clock and personalized greeting
- Asks you "What is your main focus today?" every morning
- Keeps your focus visible all day - resets the next day automatically
- Simple task list - add, complete, delete
- Focus Mode - blocks Instagram, YouTube, LinkedIn, Reddit and more
- Shows your own focus message instead of an error on blocked sites
- Settings panel - change your name, upload wallpaper, manage block list

## How I built it - My exact process
Idea → PRD → Build → Debug → GitHub

**Step 1 - Idea**
I wanted a distraction-free new tab that keeps me focused on what matters.

**Step 2 - PRD (Product Requirements Document)**
I used **Claude AI** to write a detailed PRD breaking the project into 4 phases. 
This became my blueprint. Every feature, every edge case, all documented before 
writing a single line.

**Step 3 - Build**
I used **Antigravity** (an AI coding tool) and fed it the PRD phase by phase. 
It generated all the code. I reviewed, tested, and debugged.

**Step 4 - Ship**
Loaded it as an unpacked Chrome extension. Fixed real errors. Made UI tweaks. 
Pushed to GitHub.
